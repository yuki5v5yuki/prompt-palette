# Privacy Policy - Prompt Palette

Last updated: 2026-04-01

## Overview

Prompt Palette is a desktop application for managing and quickly launching AI prompts. This privacy policy explains how the application handles your data.

## Data Collection

Prompt Palette does **not** collect, transmit, or share any personal data. All data is stored locally on your device.

## Local Data Storage

The application stores the following data locally on your computer:

- **Prompt templates**: The prompts you create and manage
- **Application settings**: Your preferences (theme, font size, hotkey, language, etc.)

All data is stored in your system's application data directory and is never sent to external servers.

## Network Access

Prompt Palette does **not** make any network requests. The application works entirely offline.

## Third-Party Services

Prompt Palette does **not** use any third-party analytics, tracking, or advertising services.

## Children's Privacy

Prompt Palette does not collect any data from anyone, including children under 13.

## Changes to This Policy

If this privacy policy changes, the updated version will be published in the application's GitHub repository.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/yuki5v5yuki/prompt-palette).
